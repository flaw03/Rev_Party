Lit un fichier CSV contenant les votes sous la forme de bulletins et les transforme en matrice de duels.

Cette fonction prend en entrée le nom d'un fichier CSV contenant les bulletins de vote, puis elle crée une matrice de duels représentant les comparaisons deux à deux des candidats. La matrice résultante est stockée dans une structure Matrice.

Parameters
filenameLe nom du fichier CSV contenant les bulletins de vote.
MatriceUn pointeur vers la structure Matrice pour stocker les données lues.
Returns
Le nombre d'électeurs dans le fichier.
